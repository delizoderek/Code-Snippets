CSS Cheatsheet

Checkout these useful CSS sheets

Linear Gradient

Add some style to your backgrounds with a linear gradient


    .box {
        background: linear-gradient(
            -10deg,
            rgb(201, 107, 123) 0%,
            maroon 50%,
            black 100%
        );
    }

Rotate on Hover

A good website always has cool transitions


    #rotate-on-hover:hover{
        transition: transform 1s ease;
        transform: rotate(360deg);
    }
